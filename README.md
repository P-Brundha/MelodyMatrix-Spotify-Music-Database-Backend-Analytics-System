# MelodyMatrix – Spotify Music Database Backend & Analytics System

MelodyMatrix is a backend-driven music data platform inspired by Spotify, designed to store, process, and serve large-scale artist and track data efficiently.

## 🚀 Features
- MySQL-based normalized database for artists and tracks
- Python ETL pipeline for data cleaning, transformation, and batch loading
- Flask REST APIs for real-time data access
- Search, filtering, and pagination for scalable queries
- Popularity-based analytics and insights

## 🛠️ Tech Stack
- Python (Pandas, MySQL Connector)
- MySQL
- Flask (REST API)
- CSV Data Processing

## 📊 Functionality
- Retrieve artists and tracks with pagination
- Search by name and filter by popularity
- Efficient batch data insertion
- Real-time query handling through APIs

## 📌 Use Case
This project demonstrates backend development, data engineering, and database management concepts by building a scalable Spotify-like music analytics system.
